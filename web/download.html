安裝說明

Windows 安裝步驟

  1. 下載對應的 .exe 安裝檔案
  2. 以系統管理員身份執行安裝程式
  3. 按照安裝精靈完成安裝
  4. 首次啟動時輸入您的 HiveMind 帳號資訊
  5. 配置節點參數並開始運行

macOS 安裝步驟

  1. 下載 .dmg 檔案並開啟
  2. 將 HiveMind 拖拽到 Applications 資料夾
  3. 首次執行時允許來自未識別開發者的應用程式
  4. 輸入您的 HiveMind 帳號資訊
  5. 配置節點參數並開始運行

Linux 安裝步驟

  1. Ubuntu/Debian: sudo dpkg -i hivemind-*.deb
  2. CentOS/RHEL: sudo rpm -i hivemind-*.rpm
  3. 或解壓 .tar.gz 並執行 install.sh
  4. 執行 hivemind --login 登入帳號
  5. 使用 hivemind --start 啟動節點

Android 安裝步驟

  1. 開啟「設定」->「安全性」->「未知來源」
  2. 下載並安裝 .apk 檔案
  3. 開啟 HiveMind 應用程式
  4. 登入您的 HiveMind 帳號
  5. 設定後台運行權限並開始挖礦

  • 2GB 可用磁碟空間

  • macOS 11.0 或更高版本
  • 4GB RAM 或以上
  • 2GB 可用磁碟空間
  • Intel 或 Apple Silicon

Linux

  • Ubuntu 18.04+ / CentOS 7+
  • 4GB RAM 或以上
  • 2GB 可用磁碟空間
  • Docker 支援

Android

  • Android 8.0 或更高版本
  • 3GB RAM 或以上
  • 1GB 可用儲存空間
  • 後台運行權限
