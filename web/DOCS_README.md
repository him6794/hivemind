# 📚 HiveMind 文檔使用指南

## 如何訪問新的文檔系統

### 1. 啟動 Web 服務
```bash
cd web
python app.py
```

### 2. 訪問文檔
在瀏覽器中訪問以下地址：

- **文檔首頁：** http://localhost:5000/docs/zh/

## 🔧 開發模式

如果你要修改文檔內容：

1. **編輯模板：** 文檔模板位於 `web/templates/docs/zh/`
2. **修改樣式：** CSS 樣式直接嵌入在各模板中
3. **添加路由：** 在 `web/app.py` 中添加新的文檔路由
4. **重啟服務：** 修改後重啟 Flask 應用
